# Vortex Browser

Ein moderner Browser mit Dark Mode und Tab-Switching-Funktionalität.

## Installation

Die Installation der erforderlichen Pakete:

```
pip install -r requirements.txt
```

## Starten

So startest du den Browser:

```
python main.py
```

## Hinweise

Der Browser verwendet PyQt5 und PyQtWebEngine, um eine moderne Benutzeroberfläche mit Webbrowser-Funktionalität zu bieten.

Funktionen:
- Modernes Dark Mode Design
- Mehrere Tabs mit Tab-Switching
- Navigationsleiste mit vor/zurück/neu laden/home Buttons
- URL-Eingabefeld
